# Smart Task Management System

This is a MERN stack project to manage tasks. Users can register, login, and manage their own tasks.

## Features
- User registration and login
- JWT-based authentication
- Add, edit, delete tasks
- Filter, search, and sort tasks
- Soft delete tasks

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB

## Installation & Run

### Backend
1. Go to backend folder:
cd backend
2. Install dependencies:
npm install
3. Start server:
npx nodemon server.js
### Frontend
1. Go to frontend folder:
cd frontend
2. Install dependencies:

npm install

3. Start app:
npm start

## Usage
- Open browser at `http://localhost:3000`
- Register → Login → Manage your tasks
